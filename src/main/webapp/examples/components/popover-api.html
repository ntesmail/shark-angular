> 配置项： >> close - 是否必须：否 - 类型：string - 默认值：'targetclick' - 说明：关闭方式，targetclick 点击触发的按钮关闭；bodyclick 点击body任意部位关闭 >> direction - 是否必须：否 - 类型：string - 默认值：'right' - 说明：优先弹出方向，top right bottom left（如果left/top空间不够，会变为right/bottom） >> name - 是否必须：否 - 类型：string - 默认值：无 - 说明：如果定义了name属性，把popover组件赋给$scope，比如name为popover1，那么controller中就可以使用 $scope.popover1对象，$scope.popover1对象所具有的方法请参见 - shark-ui的api >> popover-content - 是否必须：否 - 类型：string - 默认值：'' - 说明：内容信息 >> popover-content-url - 是否必须：否 - 类型：string - 默认值：'' - 说明：内容的地址 >> popover-title - 是否必须：否 - 类型：string - 默认值：'' - 说明：头部信息
