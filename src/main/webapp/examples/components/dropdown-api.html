> 配置项： >> data - 是否必须：是 - 类型：array - 默认值：[] - 说明：下拉框数据，eg： [{ value: 'create', name: '新建' }, { value: 'edit', name: '编辑' }, { value: 'delete', name: '删除' }] >> text - 是否必须：是 - 类型：string - 默认值：'' - 说明：展示值 >> actual-key - 是否必须：否 - 类型：string - 默认值：'value' - 说明：对应data中的真值字段 >> display-key - 是否必须：否 - 类型：string - 默认值：'name' - 说明：对应data中的展示值字段 >> on-selected - 是否必须：否 - 类型：function - 默认值：function(itemData) {} - 说明：点击下拉菜单选项之后的回调函数
