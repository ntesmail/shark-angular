> 配置项： >> tooltip-content - 是否必须：tooltip-content 或 tooltip-content-url 必须有一个 - 类型：string - 默认值：'' - 说明：tooltip内容模板 >> tooltip-content-url - 是否必须：tooltip-content 或 tooltip-content-url 必须有一个 - 类型：string - 默认值：'' - 说明：tooltip内容模板引用的url >> direction - 是否必须：否 - 类型：string - 默认值：'right' - 说明：优先弹出方向，top right bottom left（如果left/top空间不够，会变为right/bottom）
