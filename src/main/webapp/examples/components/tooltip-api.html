> 配置项： >> direction - 是否必须：否 - 类型：string - 默认值：'right' - 说明：优先弹出方向，top right bottom left（如果left/top空间不够，会变为right/bottom） >> name - 是否必须：否 - 类型：string - 默认值：无 - 说明：如果定义了name属性，把tooltip组件赋给$scope，比如name为tooltip1，那么controller中就可以使用 $scope.tooltip1对象，$scope.tooltip1对象所具有的方法请参见 - shark-ui的api >> tooltip-content - 是否必须：否 - 类型：string - 默认值：'' - 说明：内容信息 >> tooltip-content-url - 是否必须：否 - 类型：string - 默认值：'' - 说明：内容的地址
