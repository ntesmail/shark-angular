> 配置项： >> popover-title - 是否必须：是 - 类型：string - 默认值：'' - 说明：头部模板 >> popover-content - 是否必须：popover-content 或 popover-content-url 必须有一个 - 类型：string - 默认值：'' - 说明：头部模板 >> popover-content-url - 是否必须：popover-content 或 popover-content-url 必须有一个 - 类型：string - 默认值：'' - 说明：头部模板的引用url >> close - 是否必须：否 - 类型：string - 默认值：'targetclick' - 说明：关闭方式，targetclick 点击触发的按钮关闭；bodyclick 点击body任意部位关闭 >> direction - 是否必须：否 - 类型：string - 默认值：'right' - 说明：优先弹出方向，top right bottom left（如果left/top空间不够，会变为right/bottom）
